# P39. 문서 업데이트 계획 (v0.0.5 반영)

- **날짜**: 2026-02-26
- **유형**: 계획 (Plan)
- **대상 버전**: 0.0.5

## 배경

v0.0.5에서 Raw Data 패널, Similarity Index 4종, NMDS 패널, Undo/Redo, 패널 토글 등 주요 기능이 대폭 추가되었으나, 사용자 문서(README, USER_MANUAL, Sphinx docs)가 v0.0.3~2.0 시점에서 멈춰 있음. 버전 번호, 날짜, 화면 구성, 워크플로우 설명이 모두 현재 상태와 불일치.

## 수정 대상 및 내용

### 1. README.md

- **버전 수정**: 402행 `0.0.3` → `0.0.5`
- **주요 기능 섹션 업데이트**:
  - Raw Data 입력 (Presence/Absence Matrix)
  - Similarity Index 4종 (Jaccard, Ochiai, Raup-Crick, Simpson)
  - NMDS 시각화 (2D/3D)
  - Undo/Redo
  - .accdata 프로젝트 파일 지원
- **의존성 목록에 scikit-learn 추가** (98행 부근)
- **사용법 섹션**: 3단계 → 확장된 워크플로우 (Raw Data → 유사도 계산 → ACC/NMDS)
- **기술 스택**: Python 3.8+ → 3.11+

### 2. doc/USER_MANUAL.md

- **헤더**: 버전 2.0 → 0.0.5, 날짜 → 2026-02-26
- **1.2 주요 특징**: 5패널 구성, Raw Data, NMDS 추가
- **3.2 화면 구성**: 3열 → 5패널 레이아웃 (Data | Similarity | Dendrogram | ACC | NMDS)
- **새 섹션 추가**:
  - Raw Data 입력 (Presence/Absence Matrix, 시트 관리, CSV 가져오기)
  - 유사도 계산 (4종 Similarity Index, Raup-Crick iterations)
  - NMDS 분석 (2D/3D, Stress 해석)
  - 패널 관리 (View 메뉴, 패널 토글)
  - Undo/Redo (Ctrl+Z/Ctrl+Y)
  - ACC2 옵션 (Min/Max Diameter, ACC1 Style, Limit Angle)
- **6. 데이터 형식**: .accdata, Presence/Absence CSV 형식 추가
- **B. 키보드 단축키**: 실제 동작 단축키 업데이트
- **변경 이력**: v0.0.5 항목 추가
