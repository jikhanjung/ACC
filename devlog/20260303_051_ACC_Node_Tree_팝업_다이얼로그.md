# 051 — ACC Node Tree 팝업 다이얼로그 구현

**날짜:** 2026-03-03
**유형:** 작업 로그

## 작업 내용

ACC를 생성할 때 `build_acc_tree()`가 빌드한 ACCNode 트리를 클라도그램(cladogram) 스타일 덴드로그램으로 보여주는 비모달(non-modal) 팝업 윈도우를 구현했다.

### 추가된 함수/클래스

**`_draw_acc_node_tree(ax, root, title="")`** (`acc_gui.py`):
- In-order traversal로 leaf에 x=0,1,...n-1 배정
- Internal node: x = (left_x + right_x) / 2, y = node.local_sim
- 검은 실선으로 수직/수평 엣지 연결
- Leaf: 검은 원 + 흰색 area 이름 텍스트
- Internal node에 파란 박스(sim_local)와 빨간 박스(sim_global) 표시
- y축 = Similarity(0~1), x축 = leaf 순서

**`class ACCNodeTreeDialog(QDialog)`** (`acc_gui.py`):
- `QDialog`를 상속한 비모달 윈도우
- `Qt.Window` 플래그로 독립 윈도우로 동작
- matplotlib `FigureCanvas` + `_draw_acc_node_tree()` 사용
- 크기: 520×420

### generate_acc() 수정

`self.acc_tree_root` 설정 후 자동으로 `ACCNodeTreeDialog`를 열도록 수정:
```python
self.acc_tree_dialog = ACCNodeTreeDialog(
    self.acc_tree_root, title=tree_title, parent=self
)
self.acc_tree_dialog.show()
```
- 기존 다이얼로그가 열려 있으면 먼저 닫고 새로 열기
- `self._current_file`에서 파일명(확장자 제외)을 제목으로 사용
- 다이얼로그 실패 시 경고 로그만 남기고 ACC 생성은 계속 진행
