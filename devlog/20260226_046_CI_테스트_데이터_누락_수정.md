# 046. CI 테스트 데이터 파일 누락 수정

- **날짜**: 2026-02-26
- **유형**: bugfix
- **커밋**: 32ff405

## 문제

GitHub Actions CI에서 2개 테스트가 `FileNotFoundError`로 실패:

```
ERROR tests/integration/test_pipeline.py::TestFullPipeline::test_sample_data_pipeline
ERROR tests/unit/test_acc_utils.py::TestDictMatrixFromDataframe::test_dataframe_conversion
FileNotFoundError: No such file or directory: 'tests/test_data/sample_local.csv'
```

## 원인

`conftest.py`의 `sample_local_df`, `sample_global_df` fixture가 `tests/test_data/` 경로에서 CSV를 로드하도록 되어 있으나, 해당 디렉토리에 `sample_local.csv`와 `sample_global.csv`가 존재하지 않았음.

실제 파일은 프로젝트 루트의 `data/` 디렉토리에만 존재.

## 해결

`data/sample_local.csv`, `data/sample_global.csv`를 `tests/test_data/`로 복사하여 추가.

## 영향 범위

- `tests/conftest.py` — `sample_local_df`, `sample_global_df` fixture
- `tests/integration/test_pipeline.py` — `test_sample_data_pipeline`
- `tests/unit/test_acc_utils.py` — `test_dataframe_conversion`

## 결과

69 passed, 2 errors → **71 passed, 0 errors**
