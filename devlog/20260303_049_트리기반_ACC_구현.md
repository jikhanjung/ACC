## 상세

### acc_core_tree.py (~290줄)

**ACCNode 데이터클래스:**
- `members`, `left`, `right`, `local_sim`, `global_sim`, `diameter`, `angle`, `merge_order`, `diversity`, `points`
- `is_leaf`, `leftmost_leaf`, `rightmost_leaf`, `radius` 프로퍼티

**핵심 함수:**
- `build_acc_tree()`: greedy agglomeration으로 ACCNode 트리 구축
- `_determine_order()`: diversity index 기반 left/right 결정
  - 두 leaf: diversity 높은 쪽이 left, 동률이면 ABC 순
  - 클러스터 포함: edge similarity 비교로 방향 결정
- `render_tree()`: 좌표 계산 (별도 단계, 트리 재구축 불필요)
- `generate_steps()`: GUI용 progressive step snapshot 생성
- `build_acc_from_tree()`: 최상위 API (build + render + steps)

**좌표 규약:** acc_core_new.py와 동일 (pol2cart: 0°=north, +90 rotation)
