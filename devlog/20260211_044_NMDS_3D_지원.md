# 044: NMDS 3D 지원 — 2D/3D 콤보박스 전환

**작업 일시**: 2026-02-11

## 요약
NMDS 패널에 2D/3D 차원 선택 콤보박스를 추가하여, 기존 2D 산점도 외에 3D 산점도도 생성할 수 있도록 구현.

## 변경 내용

### 1. NMDSPanel.setup_content() — Dim 콤보박스 추가

Matrix 콤보박스 옆에 "Dim:" 라벨과 `2D`/`3D` 콤보박스 추가.

```python
controls_layout.addWidget(QLabel("Dim:"))
self.dim_combo = QComboBox()
self.dim_combo.addItems(["2D", "3D"])
```

### 2. MainWindow.run_nmds() — n_components 전달

콤보 선택값에 따라 `n_components`를 2 또는 3으로 전달.

```python
n_components = 3 if self.nmds_panel.dim_combo.currentText() == "3D" else 2
self.nmds_panel.nmds_widget.run_nmds(similarity_dict, n_components=n_components)
```

### 3. NMDSVisualizationWidget.run_nmds() — 2D/3D 분기 처리

- `n_components` 파라미터 추가 (기본값 2)
- MDS 호출 시 `n_components` 반영
- **3D**: `projection="3d"` axes, `ax.scatter` 3축, `ax.text` 라벨, `set_zlabel`
- **2D**: 기존 로직 유지 (scatter, annotate, axhline/axvline, equal aspect)
- info_label에 "2D" 또는 "3D" 표시 추가

- `acc_gui.py`

## 검증 방법
- Dim 콤보에서 "2D" 선택 후 Run NMDS → 기존 2D 산점도 확인
- Dim 콤보에서 "3D" 선택 후 Run NMDS → 3D 산점도, 마우스 드래그 회전/줌 확인
