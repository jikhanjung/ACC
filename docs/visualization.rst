시각화 해석
===========

알고리즘 이해
--------------

ACC는 다음 공식을 사용합니다:

Diameter (지름)
~~~~~~~~~~~~~~~

.. code-block:: text

   d = 1 + (1 - sim_global)

* **sim_global가 높을수록** → d가 작음 → **중심에 가까움**
* **sim_global가 낮을수록** → d가 큼 → **바깥쪽에 위치**

예시:

* sim_global = 0.9 → d = 1.1 (중심 근처)
* sim_global = 0.5 → d = 1.5 (바깥쪽)

Angle (각도)
~~~~~~~~~~~~

.. code-block:: text

   θ = 180° × (1 - sim_local)

* **sim_local가 높을수록** → θ가 작음 → **가까운 각도**
* **sim_local가 낮을수록** → θ가 큼 → **먼 각도**

예시:

* sim_local = 0.9 → θ = 18° (가까움)
* sim_local = 0.5 → θ = 90° (직각)

시각화 패턴 읽기
----------------

패턴 1: 타이트한 클러스터
~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: text

       J
      / \
     T   Y

**특징**:

* 작은 각도 차이
* 동일한 원 위에 위치
* 높은 local similarity

**해석**: J, T, Y는 매우 유사한 그룹

패턴 2: 계층적 구조
~~~~~~~~~~~~~~~~~~~~

.. code-block:: text

            Root
           /    \
       Group1  Group2
       /  \      /  \
      J    T    N    O

**특징**:

* 여러 레벨의 동심원
* 안쪽 → 바깥쪽으로 계층 형성

**해석**: 명확한 상하 계층 구조

패턴 3: 고립된 영역
~~~~~~~~~~~~~~~~~~~~

.. code-block:: text

   J T Y         Q
    ∙∙∙    →    ∙  ← 멀리 떨어짐

**특징**:

* 큰 각도 차이
* 다른 원에 위치

**해석**: Q는 다른 영역들과 유사도가 낮음

두 Similarity의 역할
---------------------

Local Similarity
~~~~~~~~~~~~~~~~~~~~~~~

* **역할**: 같은 레벨 내에서의 상대적 위치 결정
* **영향**: 각도 (angular position)
* **해석**: "얼마나 가까운가?"

Global Similarity
~~~~~~~~~~~~~~~~~~~~

* **역할**: 계층 레벨 결정
* **영향**: 반지름 (radial position)
* **해석**: "어느 레벨에 속하는가?"

예시
~~~~

영역 A, B:

* local_sim = 0.9 (높음) → 작은 각도 차이 → 가까이 위치
* global_sim = 0.5 (낮음) → 큰 반지름 → 바깥쪽 원

영역 C, D:

* local_sim = 0.5 (낮음) → 큰 각도 차이 → 멀리 위치
* global_sim = 0.9 (높음) → 작은 반지름 → 안쪽 원

시각화 해석 가이드
------------------

동심원 읽기
~~~~~~~~~~~

1. **가장 안쪽 원** (보라색): 가장 높은 global similarity를 가진 영역들
2. **중간 원들** (파란색~초록색): 중간 수준의 global similarity
3. **가장 바깥쪽 원** (빨간색): 가장 낮은 global similarity

각도 관계 읽기
~~~~~~~~~~~~~~

1. **같은 원 위의 영역들**: 동일한 global similarity level
2. **각도가 가까운 영역들**: 높은 local similarity (유사한 특성)
3. **각도가 먼 영역들**: 낮은 local similarity (다른 특성)

색상 코드 이해
~~~~~~~~~~~~~~

**동심원 색상** (무지개 colormap):

* 보라색 → 파란색 → 초록색 → 노란색 → 빨간색
* 안쪽에서 바깥쪽으로 갈수록 global similarity 감소

**Area 점 색상** (ACC1):

* 각 area의 색상은 해당 동심원의 색상과 일치
* 한눈에 어느 레벨에 속하는지 파악 가능

**하이라이트 색상**:

* 빨간색: 현재 단계에서 병합되는 클러스터
* 파란색: 병합 완료된 클러스터

실전 해석 예시
--------------

샘플 데이터 해석
~~~~~~~~~~~~~~~~

6개 영역 (J, T, Y, N, O, Q)의 경우:

1. **J, T, Y 그룹**:

   * 같은 안쪽 원에 위치 (높은 global_sim)
   * 서로 가까운 각도 (높은 local_sim)
   * → 매우 유사한 그룹

2. **N, O, Q 그룹**:

   * 같은 바깥쪽 원에 위치 (낮은 global_sim)
   * 서로 가까운 각도 (높은 local_sim)
   * → 유사하지만 JTY와는 다른 그룹

3. **두 그룹 간 관계**:

   * 다른 동심원에 위치 (다른 global_sim level)
   * 큰 각도 차이 (낮은 local_sim)
   * → 명확히 구분되는 두 클러스터
